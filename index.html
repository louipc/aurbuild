aurbuild

  • A tool to build and install packages from the Arch Linux User-Community Repository.
  • Helpful for building official packages via ABS.
  • Released under the GNU GPL version 2

Please contribute to this project! Patches are always welcome.

Installing aurbuild

pacman -U http://download.berlios.de/aurbuild/aurbuild-1.8.3-1-any.pkg.tar.gz

You can also sync aurbuild directly from pacman.
Just add these lines to your pacman.conf:

[aurbuild]
Server = http://download.berlios.de/aurbuild

History

aurbuild was first conceived as a fourty four line python script written in 2005 by Dusty Phillips which appeared in the Arch Linux wiki.

That script was then picked up by Tyler Gates. It was expanded, and even rewritten (twice). Sometime in 2007 Gates switched to another distribution yet continued to maintain aurbuild.

In February 2008 Loui Chang took over maintaining aurbuild.

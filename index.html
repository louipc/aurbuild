Maintainer Loui Chang <louipc.ist@gmail.com>

aurbuild is a tool that makes it easier to build and install packages from the Arch Linux User-Community repository.

aurbuild is also very useful for building official packages via ABS.

Please contribute to this project! Patches are always welcome.

Try it out!

You can now sync aurbuild directly from pacman.
Just add these lines to your pacman.conf:

[aurbuild]
Server = http://download.berlios.de/aurbuild
